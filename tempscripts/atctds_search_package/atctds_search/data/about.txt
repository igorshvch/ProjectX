Программа для автоматического добора судебной практики для ЭСС по налогам.
Автор: Шевченко И.С. (ЭАО, ДАПР, ПИТ). 2019 год.

ПОРЯДОК РАБОТЫ
==============
Используемые  сокращения:
Программа для автоматического добора судебной практики - ПАД
============================================================
1. Судебные акты необходимо выгрузить из суперхоста в текстовые файлы формата '<название файла>.txt'.
Для этого, в свою очередь, необходимо сделать следующее:
	1.1. При первоначальном использовании:
		1.1.1. В карточке поиска задать [Область поиска] - «Судебная практика»
		1.1.2. В карточке поиска в поле [Текст документа – Основной поиск] указать «налог инспекция»
		1.1.3. В карточке поиска в поле [Когда получен - Позже чем] указать желаемую дату, например, «01.01.2018». Рекомендуем указать более раннюю дату, чем необходима для добора по конкретным кирпичам, поскольку задать более точную дату можно внутри самой ПАД. В конечном итоге это сократит количество выгрузок судебных актов из суперхоста.
		1.1.4. После построения списка результатов выбрать соответствующий арбитражный суд округа в списке в левой области, выделить все отображенные судебные акты (Ctrl+A) в списке в правой области, кликнуть правой кнопкой мыши по выделенным заголовкам и в выпадающем списке выбрать «Сохранить в файл…».
		1.1.5. В появившемся окне указать желаемую папку для сохранения.
		1.1.6. Задать наименование файла, задать [Тип файла] – «Обычный текст без форматирования (*.txt). В области [Что сохранять в файл] выберите «Выбранные поля». Далее в списке в правой области необходимо установить «галочки» ТОЛЬКО рядок с пунктами «Когда получен» и «Текст документа». Нажать кнопку «Сохранить».
			ОБРАТИТЕ ВНИМАНИЕ: для разных окружных судов список состоит из разных элементов; для каждого окружного суда необходимо проверить, что «галочки» стояли ТОЛЬКО напротив указанных выше пунктов. В противном случае система не будет работать корректно.
		1.1.7. Для каждого арбитражного суда округа повторить пункты 1.1.4 – 1.1.6.
	1.2. При подгрузке судебных актов для обновления базы ПАД:
		1.2.1. Выполнить пункты 1.1.1 – 1.1.4
		1.2.2. В качестве папки сохранения указать
			1.2.2.1. ЛИБО папку, в которой сохранены ранее выгруженные акты;
			1.2.2.2. ЛИБО подпапку по отношению той, в которую сохранены ранее выгруженные акты.
		1.2.3. Выполнить пункты 1.1.6. – 1.1.7.
2. Заголовки кирпичей вместе с метками к ним необходимо выгрузить в текстовый файл формата «<название файла>.txt» с табуляцией в качестве разделителя:
	2.1. После того, как сформирован список кирпичей с их метками, необходимо перенести его в файл MS Excel таким образом, чтобы в одной строчке в первой ячейке находился заголовок кирпича, а во второй ячейке – метка.
	2.2. Далее в MS Excel выбрать [Файл] -> [Сохранить как] -> [Обзор]. В открывшемся окне указать желаемую папку для сохранения, задать наименование файла. Далее выбрать [Тип файла] – «Текстовые файлы (с разделителями табуляции) (*.txt)». Нажать кнопку «Сохранить». (Команды для сохранения приведены на примере MS Excel 2013, их название и порядок могут незначительно отличаться в других версиях).
3. В ПАД:
	3.1. Укажите:
		3.1.1. папку для подгрузки судебных актов в ПАД,
		3.1.2. файл для подгрузки заголовков кирпичей с метками,
		3.1.3. папки для сохранения результатов
	3.2. Нажмите кнопку «Подготовить выводы». В окне над кнопкой появится список с заголовками кирпичей и условным обозначением ЭСС, к которой они относятся. При двойном щелчке мышкой по элементу списка он полностью отобразиться в окне информации справа.
	3.3. Задайте дату, начиная с которой необходимо произвести добор.
	3.4. Нажмите кнопку «Подготовить акты». Дождитесь, пока в окне информации справа появится сообщение «Судебные акты подготовлены для добора».
	3.5. Нажмите кнопку «Сделать добор».
		3.5.1. ПАД проанализирует акты на соответствие выводам по 4 моделям.
		3.5.2. По окончании обработки выводов, относящихся к одной ЭСС, в информационном окне справа появится сообщение «Записал результаты в <путь к папке сохранения, сформированной системой на основе даты, начиная с которой производится добор>».
		3.5.3. По окончании обработки всех выводов в информационном окне справа появится сообщение «Добор завершен!».
	3.6. При необходимости провести новый добор по новой партии кирпичей в рамках текущего сеанса работы с ПАД, нажмите кнопку «Подготовить новый добор».
